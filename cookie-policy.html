Политика использования файлов cookie

1. Что такое файлы cookie

Файлы cookie — это небольшие текстовые файлы, которые сохраняются на вашем устройстве при посещении нашего сайта. Они помогают нам улучшить работу сайта и предоставить вам более персонализированный опыт.

2. Какие файлы cookie мы используем

Необходимые файлы cookie

Эти файлы cookie необходимы для корректной работы сайта:

  • Сессионные cookie для аутентификации пользователей
  • Cookie для сохранения настроек сайта
  • Cookie безопасности
Аналитические файлы cookie

Мы используем аналитические cookie для понимания того, как посетители взаимодействуют с нашим сайтом:

  • Google Analytics для сбора статистики посещаемости
  • Yandex Metrika для анализа поведения пользователей
  • Hotjar для тепловых карт и записей сессий
Функциональные файлы cookie

Эти файлы cookie позволяют сайту запоминать ваши предпочтения:

  • Настройки фильтров поиска
  • Предпочитаемый язык интерфейса
  • Сохраненные избранные объекты
Рекламные файлы cookie

Мы используем рекламные cookie для показа релевантных предложений:

  • Google Ads для ретаргетинга
  • Facebook Pixel для персонализированной рекламы
  • Yandex Direct для контекстной рекламы

3. Управление файлами cookie

Вы можете управлять файлами cookie в настройках вашего браузера:

  • Chrome: Настройки → Конфиденциальность и безопасность → Файлы cookie
  • Firefox: Настройки → Приватность и защита → Файлы cookie
  • Safari: Настройки → Конфиденциальность → Управление данными сайтов
  • Edge: Настройки → Файлы cookie и разрешения сайтов

4. Сторонние сервисы

Наш сайт использует сервисы третьих сторон, которые также могут устанавливать свои файлы cookie:

  • Google Analytics - для анализа трафика
  • Yandex Maps - для отображения карт объектов
  • Bootstrap CDN - для корректного отображения интерфейса
  • Font Awesome - для иконок интерфейса

5. Хранение данных

Файлы cookie хранятся на вашем устройстве в течение различного времени:

  • Сессионные cookie удаляются при закрытии браузера
  • Постоянные cookie хранятся до 2 лет
  • Аналитические cookie хранятся до 26 месяцев

6. Отказ от файлов cookie

Вы можете отказаться от использования файлов cookie, изменив настройки вашего браузера. Однако это может повлиять на функциональность сайта.
